Given two strings `s` and `t`, return `true` if `t` is an anagram of `s`, and `false` otherwise.

```
**Example 1:**

**Input:** s = "anagram", t = "nagaram"

**Output:** true
```

```
**Example 2:**

**Input:** s = "rat", t = "car"

**Output:** false
```

**Constraints:**

- `1 <= s.length, t.length <= 5 * 104`
- `s` and `t` consist of lowercase English letters.

**Follow up:** What if the inputs contain Unicode characters? How would you adapt your solution to such a case?

### Solución:

1. Debemos de validar si la longitud de cada palabra es igual, Si no es igual retornamos `false`
2. Creamos un map, para almacenar los caracteres recorridos de la primera palabra:
	- Si el caracter existe, debe de sumarle 1 a su valor
	- Caso contrario, debe de guardar el caracter en el map con el valor de 1
3. Recorremos la segunda palabra y validamos lo siguiente:
	- Si el caracter existe en el map y es diferente de 0, se le resta 1 a su valor
	- Si existe en el map y es igual a 0, se retorna `false`
	- Si no existe en el map, se retorna `false`
	1. En el caso de no haya ningún retorno, se devuelve true.
