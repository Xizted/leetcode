You are given two **non-empty** linked lists representing two non-negative integers. The digits are stored in **reverse order**, and each of their nodes contains a single digit. Add the two numbers and return the sum as a linked list.

You may assume the two numbers do not contain any leading zero, except the number 0 itself.

**Example 1:**

![[Pasted image 20260518171717.png]]

**Input:** l1 = [2,4,3], l2 = [5,6,4]
**Output:** [7,0,8]
**Explanation:** 342 + 465 = 807.

**Example 2:**

**Input:** l1 = [0], l2 = [0]
**Output:** [0]

**Example 3:**

**Input:** l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
**Output:** [8,9,9,9,0,0,0,1]

**Constraints:**

- The number of nodes in each linked list is in the range `[1, 100]`.
- `0 <= Node.val <= 9`
- It is guaranteed that the list represents a number that does not have leading zeros.

### Solución

1. Creamos un dummyHead para evitar validar si tenemos valor o no en la Linked List
2. Inicializamos una variable current y la igualamos con dummyHead, tambien inicializamos una variable para guardar el acarreo de las sumas
3. Hacemos un bucle validando de que l1 o l2 sean diferentes de nulo
	- Si es diferente, se debe de inicializar dos variables (a,b) para almacenar el valor de cada lista
	- Luego de valida si el siguiente nodo no es nulo, si no es nulo, se igualan l1 y l2 a su next correspondiente
	- Se realiza la operación: a + b + carry
	- Luego se obtiene el acarreo: carry / 10
	- Por ultimo el resultado del nuevo nodo: sum % 10
4. Si el carry es mayor a 0, se debe de agregar un nuevo nodo con ese valor
5. Devolver el dummyHead.next
